## 📁 هيكل المشروع

```
portfolio/
├── app.py                  # التطبيق الرئيسي (Routes + JSON API للوحة التحكم)
├── requirements.txt
├── instance/
│   └── portfolio.db        # قاعدة البيانات (تُنشأ تلقائياً إن لم تكن موجودة)

│   ├── css/
│   │   ├── style.css       # تصميم الموقع العام (Dark Tech)
│   │   └── admin.css       # تصميم لوحة التحكم (Sidebar + Dashboard)
│   └── js/
│       ├── main.js         # تفاعلات الموقع العام
│       └── admin.js        # منطق لوحة التحكم (SPA كامل، يتواصل مع /admin/api/*)
└── templates/
    ├── base.html
    ├── index.html           # الصفحة الرئيسية
    ├── project_detail.html  # تفاصيل المشروع
    ├── buy.html             # صفحة الشراء
    ├── order_success.html   # نجاح الطلب

    └── admin_login.html     # تسجيل دخول الإدارة
```

## ⚡ تشغيل المشروع

```bash
# 1. ثبّت المتطلبات
pip install -r requirements.txt

# 2. شغّل التطبيق
python app.py

# 3. افتح المتصفح على
http://localhost:5000
```

## 🔐 لوحة التحكم

- الرابط: `http://localhost:5000/admin`
- كلمة المرور الافتراضية: `admin123`
- **غيّر كلمة المرور في app.py سطر: `ADMIN_PASS = 'admin123'`**

### مميزات لوحة التحكم الجديدة

- 📊 **نظرة عامة:** بطاقات KPI (مشاريع، طلبات، إيرادات، رسائل) + رسم بياني للإيرادات آخر 14 يوم + مخطط دائري لحالة الطلبات + قوائم أحدث النشاطات
- 📁 **المشاريع:** جدول بحث وفلترة فورية (الكل / مميز / للبيع)، إضافة وتعديل وحذف عبر نوافذ منبثقة (Modal) بدون إعادة تحميل الصفحة
- 🧩 **المهارات:** شبكة بطاقات قابلة للتعديل المباشر (الاسم + المستوى بشريط Slider) مع حفظ تلقائي
- 🛒 **الطلبات:** جدول بحث وفلترة بالحالة، تغيير حالة الطلب فورياً مع إشعار Toast
- 📩 **الرسائل:** عرض بطاقات، تمييز كمقروءة/غير مقروءة، حذف
- 📱 **متجاوبة بالكامل:** الشريط الجانبي يتحول إلى قائمة منسحبة على الموبايل، وكل الجداول والبطاقات تتكيف مع الشاشات الصغيرة
- 🔔 نظام إشعارات Toast لكل عملية (نجاح / خطأ)
- 🪟 نوافذ تأكيد قبل أي حذف

كل هذه الميزات تعمل عبر API بسيط مبني تحت `/admin/api/*` يُعيد JSON، بدون أي مكتبات خارجية إضافية (Vanilla JS فقط).

## 📦 نشر على الإنترنت

للنشر على سيرفر (Render, Railway, VPS):

```bash
# أضف gunicorn
pip install gunicorn
gunicorn app:app
```

أو على Railway/Render: ارفع المجلد مباشرة وسيُنشأ تلقائياً.

## 🛡️ ملاحظة أمان

كلمة مرور الإدارة (`ADMIN_PASS`) ومفتاح الجلسة (`app.secret_key`) داخل `app.py` بشكل ثابت لأغراض التطوير المحلي. عند النشر الفعلي على الإنترنت، يُفضّل نقلهما إلى متغيرات بيئة (Environment Variables) بدلاً من كتابتهما مباشرة في الكود.
